## 🚀 快速启动指南

### 环境要求
- JDK 8+
- Node.js 16+
- MySQL 8.0+
- Redis 6+

### 启动步骤
```bash
# 1. 克隆项目
git clone <项目地址>

# 2. 数据库初始化
mysql -u root -p < sql/init.sql

# 3. 启动后端
cd campus-reading-backend
mvn spring-boot:run

# 4. 启动前端
cd campus-reading-frontend
npm install
npm run dev
```
